# Python Weather App

A modern desktop weather application built using Python and Customtkinter

## Features
- Real time weather data using Weather API
- City search with location suggestions
- Modern UI
- 5-day weather forecast

## Tech Stack
- Python 
- CustomTkinter
- Requests
- Pillow
- WeatherAPI

## How to run

1. Clone the repository
2. Install dependencies
pip instalm -r
requirements.txt
3. Run the app
